## Hacker News

Title:

```text
Show HN: Consumer Dispute Assistant, an Agent Skill for refund disputes
```

Post:

```text
I made a small Agent Skill for Claude/Codex that turns messy consumer disputes into clean case packs.

The first use case is subscription billing disputes: cancelled before renewal, still got charged.

It builds:
- a factual case summary
- an evidence timeline
- a merchant refund email
- a payment dispute statement
- a follow-up log

It is deliberately not a "robot lawyer." It does not file anything or send anything without user approval. It just does the boring part: organize the facts and draft the materials.

Repo: https://github.com/JerryNee/consumer-dispute-agent-skill
```
