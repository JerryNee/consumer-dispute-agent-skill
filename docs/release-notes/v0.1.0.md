# v0.1.0 - Subscription Dispute Case Packs

Initial public release of `consumer-dispute-assistant`.

## Included

- Agent Skill for Claude, Codex, and Agent Skills-compatible clients.
- Workflow for subscription cancellation failures, duplicate charges, refund denials, and basic complaint escalation.
- Evidence checklist for receipts, cancellation screenshots, support logs, billing records, and missing facts.
- Escalation playbooks for merchant support, payment disputes, BBB/regulator drafts, and sector-specific routes.
- Safety boundaries for legal-adjacent, financial, fraud, and high-conflict cases.
- Markdown templates for:
  - case summary
  - merchant refund request
  - payment dispute statement
  - regulator complaint draft
  - follow-up log
- `build_case_pack.py` script for generating a complete Markdown case pack from JSON.
- Example subscription double-charge intake.

## Philosophy

This is not a robot lawyer. The skill helps agents organize facts and draft materials, while preserving a user confirmation gate before external action.

## Try It

```bash
python3 skills/consumer-dispute-assistant/scripts/build_case_pack.py \
  examples/subscription-double-charge.json \
  --out dispute-pack \
  --tone firm \
  --today 2026-06-18
```
